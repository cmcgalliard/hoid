# Hoid

> *"The most important step a man can take. It's not the first one, is it? It's the next one."*

Hoid is my personal agentic AI skills bot — a collection of skills, workflows, and personalities designed to power intelligent, context-aware AI agents across coding sessions.

## What is Hoid?

Named after the mysterious, ever-present character from Brandon Sanderson's Cosmere, Hoid shows up wherever he's needed, adapts to any situation, and always has a role to play. This repo is a set of skills and personalities that give AI agents that same quality: flexible, purposeful, and effective.

The repo contains:

- **`skills/`** — Reusable workflow skills (brainstorming, code review, state management, team orchestration)
- **`personalities/`** — Personality definitions that shape how an agent thinks and communicates
